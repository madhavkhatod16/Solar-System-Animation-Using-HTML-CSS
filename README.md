# Solar-System-Animation-Using-HTML-CSS
A visually appealing Solar System simulation built using pure HTML and CSS. Features a glowing Sun, orbiting planets, realistic space background, smooth animations using CSS transforms, and responsive design. No JavaScript is used.
A Solar System animation built with HTML and CSS featuring the Sun, Mercury, Venus, Earth, Mars, Jupiter, Saturn, Uranus, and Neptune orbiting in space with smooth CSS animations and realistic visual effects.
